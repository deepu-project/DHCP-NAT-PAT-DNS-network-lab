# Enterprise Network Lab

## 📌 Overview
This project is a multi-site enterprise network designed using Cisco Packet Tracer. It simulates real-world network architecture with multiple locations, routing, and IP management.

## 🛠️ Technologies Used
- OSPF Routing
- NAT / PAT
- DHCP
- Subnetting

## 🌐 Network Design
- Multi-site topology (e.g., Delhi, Mumbai)
- Interconnected routers using OSPF
- NAT / PAT configuration
- Structured IP addressing (/27, /30)

## 🔧 Features
- Dynamic routing using OSPF
- NAT/PAT for external communication
- DHCP for automatic IP allocation
- Network troubleshooting and testing
